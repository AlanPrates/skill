- **Design Moderno** - Glassmorphism, gradientes e animações suaves
- **Dragon Fire Mini Game** - Mini game interativo integrado via Canvas

## 🎮 Dragon Fire Mini Game

Um mini game de combate onde você controla um dragão para enfrentar ondas de inimigos mecânicos.

- **Controle**: Clique e arraste para mover o dragão.
- **Ataque**: Pressione a tecla `1` para disparar bolas de fogo.
- **Score**: Sistema de pontuação em tempo real.
- **Áudio**: Efeitos sonoros imersivos.

## 🛠️ Tecnologias

- **Frontend**: HTML5 Semântico, CSS3 (Variables, Grid, Flexbox)
- **Engine de Game**: JavaScript Vanilla (Canvas API)
- **Fontes**: Google Fonts (Inter, Outfit)
- **Deployment**: Compatível com qualquer servidor estático

## 🚀 Como Usar

Abra `index.html` no navegador ou sirva com qualquer servidor HTTP:

```bash

```
